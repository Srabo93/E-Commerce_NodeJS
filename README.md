# E-Commerce_NodeJS

Simple E-commerce Shop

### Dev Setup

```

npm run container

npm install

npm run dev



database seeder: node config/seeder.js

```

.env

```

PORT=
SESSION_SECRET=
TOKEN_SECRET=

SMTP_HOST=
SMTP_PORT=
SMTP_USER=
SMTP_PASSWORD=

FROM_NAME=
FROM_EMAIL=
PSQL_URI=
NODE_ENV=development
```
